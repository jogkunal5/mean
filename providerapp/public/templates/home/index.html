Provider App

"Provider app is a demo project to show case the MEAN technologies using excel file structures as a first step"

What ProviderApp does?

A web application which picks all the provider details exported to an excel sheet into the database

Display the primary fields data of the provider as a first look-up

User can edit any provider data just clicking on edit

Once the edit is done the updated providers list can be exported to excel and is ready to load to legacy systems
